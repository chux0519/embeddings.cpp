Snowflake Browser WASM

Run the quantized Snowflake model in Chromium.

This page benchmarks the packaged q4_k_mlp_q8_attn artifact in Chromium with the same token batch across single-thread WASM, pthread WASM, and WebGPU when the browser exposes a compatible adapter.
